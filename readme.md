# VT Chat
## What is this
This is a chat made by @VertexStudioOfficial and @OnlyAHuman using PHP and Server-Sent Events(SSE).
## New Features
- Server-sent events, much better than fetch polling
- Nice and simple page layout
- More backgrounds
- Reply and login system
